安装ZSH

2018-10-25 shelloh-my-zsh 阅读量

Zsh?

Oh-My-Zsh是Zsh的框架，Z shell。

  • 为了让Oh-My-Zsh工作，必须安装Zsh。

    • 请运行zsh --version确认。
    • 预期结果:zsh 5.1.1或更近期

  • 另外，Zsh应该被设置为您的默认shell。

    • 请从一个新的终端运行echo $SHELL来确认。
    • 预期结果:usr/bin/zsh或类似

安装并设置zsh为默认值

如有必要，请按照以下步骤安装Zsh:

  1. 安装Zsh有两种主要方法

    • 使用您选择的包管理器，例如sudo apt-get install zsh(请参阅下面的示例)
    • 来自source,以下是Zsh FAQ中的说明

  2. 通过运行zsh -version来验证安装。预期结果:zsh 5.1.1或更近期。

  3. 让它成为您的默认shell: chsh -s $(which zsh)

    • 注意，如果Zsh不在您的授权shell列表(/etc/shell)中，或者您没有使用chsh的权限，那么这将不起作用。如果是这样的话，你需要使用不同的程序

  4. 注销并再次登录以使用新的默认shell。

  5. 测试它是否工作 echo $SHELL。预期结果:/bin/zsh或类似。

  6. 用 $SHELL --version测试。预期结果:“zsh 5.1.1”或类似

如何在多个平台上安装zsh

macOS

用Homebrew安装之前尝试zsh --version。如果它比4.3.9更新，你可能就没问题了。最好比5.0更新或等于5.0
